# library-api
Criação de uma API RestFull que busca , cria , atualiza e deleta livros na base de dados

Ferramentas usadas:

# Spring-Boot-Web
# Spring-Boot-JPA
# Spring-Boot-Mail
# Spring-Boot-Actuator
# Spring-Boot-Admin
# Spring-Boot-DevTools
# H2 Databa Base
# Lombok
# ModelMapper
# Swagger
# Junit5
# Mockito
# Jacoco
